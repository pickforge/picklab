# PickLab

PickLab gives AI coding agents eyes, hands, and a reproducible lab for native apps — think Playwright for native apps and Android emulators. The MVP covers Linux desktop sessions (Xvfb/VNC, window, input, and screenshot automation) and Android emulator orchestration (AVD, ADB, UIAutomator, logcat), exposed through the `picklab` CLI and an MCP server published as `@pickforge/picklab`.
